# mini.

A minimalist, high-performance front-end code minifier built for developers who want fast, clean, and distraction-free optimization.

mini. compresses and minimizes HTML, CSS, JavaScript, and JSON directly in the browser with a polished desktop-focused interface and real-time workflow enhancements.

---

## Features

- HTML minification
- CSS compression
- JavaScript minification via Terser
- JSON minification
- Live minification mode
- One-click clipboard paste/copy
- Real-time character statistics
- Compression savings indicator
- Comment stripping
- Whitespace collapsing
- Optional JS variable mangling
- Fully client-side processing
- Modern minimalist UI
- Responsive desktop-first layout
- Smooth animations and transitions
- Zero backend required

## Interface Overview

### Input Panel
Paste or type your source code into the editor.

### Output Panel
View the optimized result instantly with live statistics.

### Live Mode
Automatically minifies code while typing.

### Settings Panel
Customize optimization behavior with advanced toggles:
- Strip comments
- Collapse whitespace
- Mangle JavaScript variables
